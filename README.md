# RETLflow

<!-- badges: start -->

<!-- badges: end -->

RETLflow package integrates RETL and Rflow packages into a seamles ETL
workflow in R.

## Installation

`RETLflow` can be installed from [GitHub](https://github.com/vh-d/RETL)
by running:

``` r
devtools::install_github("vh-d/RETLflow")
```
